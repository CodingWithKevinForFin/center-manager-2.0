Set Class Overview

Definition

Set var = .... ;

Description

A mutable collection of unique data entries. Attempts to add duplicates will be ignored. Entries are stored in the order inserted (Backed by Java LinkedHashSet)

Method Definitions

<Constructor> Returns true if the set contains ANY of the supplied values
iterator returns an iterator over the entries
contains Returns true if the set contains ANY of the supplied values
size Get the size of this set
addAll adds all elements from provided list
venn Returns a new set of values which is a combination of this set and the other set. Consider the two sets comprising a venn diagram and the three boolean flags determining which portions of the diagram are included in the final set
add Adds all supplied values if they don't already exist. Returns true if the set changed as a result of this call
remove Remove all supplied values if they exist. Returns true if the set changed as a result of this call
