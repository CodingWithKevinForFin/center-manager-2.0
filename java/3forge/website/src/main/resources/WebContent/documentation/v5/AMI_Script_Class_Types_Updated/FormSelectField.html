FormSelectField Class Overview

Definition

FormSelectField var = .... ;

Description

A special type of field within a FormPanel that allows multiple options as a dropdown

Method Definitions

resetPosition Reset field position to default position specified in field editor.
removeOption Adds an option to the multi-select field.
setOptions Set the field's options using the provided map
isAtDefaultPosition Get whether field is at the default position specified in field editor.
putOption Adds an option to the multi-select field.
getOptions Returns the map of options
getDisplayMode Get display type of field. Will return "ABSOLUTE" if field is displayed using absolute positioning. Will return "HTML" if field is displayed using form HTML
getOption Returns a single option according to input key
clearOptions Removes all options.
setPosition Set field position. Specify (x,y) position, width, and height in pixels. Passing null to an argument is equivalent to not setting a value for that specific argument.
getFieldValue Return the value of the field with the associated varName for the form that owns this field.
setDisplayMode Set display mode of field. Setting to "ABSOLUTE" will display field using absolute positioning. Setting to "HTML" will allow field to be displayed using the form HTML. Any other input will throw an exception.
getPanel Return the panel that contains this field.
getValue Return the value associated with this field.
getPosition Get a Rect object representing the field's position as (x,y,w,h).
setValue Set this field's value.
addOptions Add to the field's options based on the provided map
