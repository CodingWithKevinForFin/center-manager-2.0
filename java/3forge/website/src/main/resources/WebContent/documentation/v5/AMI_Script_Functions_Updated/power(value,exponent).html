Definition

power(number value, number exponent)

Description

Returns the value of the first argument raised to the power of the exponent

Return Value

double value

Parameter Definition

number value value to raise to exponent
number exponent exponent to raise value to

Examples

double r1 = power(10,1); // r1 == 10.0
double r2 = power(10,0); // r2 == 1.0
double r3 = power(10,3); // r3 == 1000.0
double r4 = power(5.3D,4.2D); // r4 == 1101.434440782885
