Definition

switch(Object key, Object defaultValue, Object ... keyValueChoices)

Description

Takes a mapping of key/value pairs and then given a supplied key returns the associated value, or a defaultValue if the supplied key does not existing in the provided mapping

Return Value

Either the defaultValue or one of the keyValueChoices' values

Parameter Definition

Object key the key used to look up a value in the map
Object defaultValue the defaultValue to return if the key is not found in supplied keyValueChoices
Object ... keyValueChoices An even number of arguments ordered such that: key1,value2,key2,value2,.... KEYS MUST EVALUATE TO CONSTANT EXPRESSIONS AND BE UNIQUE

Examples

String r1 = switch(0,"NA",0,"ZERO",1,"ONE",2,"TWO",null,"NULL"); // r1 == ZERO
String r2 = switch(2,"NA",0,"ZERO",1,"ONE",2,"TWO",null,"NULL"); // r2 == TWO
String r3 = switch(7,"NA",0,"ZERO",1,"ONE",2,"TWO",null,"NULL"); // r3 == NA
String r4 = switch(null,"NA",0,"ZERO",1,"ONE",2,"TWO",null,"NULL"); // r4 == NULL
