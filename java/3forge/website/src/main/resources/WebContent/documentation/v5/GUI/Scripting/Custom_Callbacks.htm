Custom Callbacks

Objects can have AmiScript attached to them for responding to certain key events, such as resize events or certain user actions, etc. (see Green Button > AmiScript Callbacks). Additionally, there are dashboard-level callbacks such as when the dashboard is loaded, see Menu > Dashboard > Custom Callbacks.
