Relationships

Relationships are a fast and easy way to link certain user actions on one panel to a target datasource or target realtime panel. User actions can include changes to selection, double clicking, or invoking a context menu item. There are two types of relationships, depending on the target type:

 

Real time Relationships

These target realtime panels. The relationship uses the selected input data to construct a live query that is sent to the target real time panel.

 

Static Relationships

These target datamodels or blenders. The relationship uses the selected input data to construct a set of variables that are sent to the target datamodel.
