Definition

formatNumber(number value, String format, String options)

Description

Converts a numeric value to string

Return Value

String, Formatted number based on supplied format

Parameter Definition

number value value to convert to string
String format pattern for converting value. See java DecimalFormat for details
String options Place holder, leave as empty string

Examples

String r1 = formatNumber(123.456D,"#",""); // r1 == 123
String r2 = formatNumber(123.456D,"#,###",""); // r2 == 123
String r3 = formatNumber(123.456D,"#,###.000",""); // r3 == 123.456
String r4 = formatNumber(123.456D,"#,###.E0",""); // r4 == 123.5E0
