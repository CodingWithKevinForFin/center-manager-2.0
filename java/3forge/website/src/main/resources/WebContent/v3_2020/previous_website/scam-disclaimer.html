Scam Prevention Disclaimer

How to Spot Online Scams

3forge takes information security seriously and here are some helpful tips to identify scammers:

  • 3forge will never request payments through e-commerce websites.
    If you are requested to send money to 3forge via an e-commerce website, it is a scam.
  • 3forge official materials are only found at 3forge.com and associated sub domains.
    If you receive links to sites that present themselves as 3forge but are not under the 3forge.com domain, it is a scam.
  • 3forge email contacts will always have 3forge.com in the domain.
    If you receive email links that present themselves as 3forge but are not under the 3forge.com domain, it is a scam.
  • 3forge does not have a mobile app.
    If you are requested to download a '3forge' app, it is a scam.

To learn more about phishing please visit: https://www.cisa.gov/secure-our-world/recognize-and-report-phishing
