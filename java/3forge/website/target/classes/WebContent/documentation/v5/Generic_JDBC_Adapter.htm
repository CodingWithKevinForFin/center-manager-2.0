Generic JDBC Adapter

Using the Generic JDBC Adapter:

 

This adapter can be used to connect to generic JDBC drivers that AMI does not have direct support for. Before using the generic driver, please contact support as there may already be a driver available for download. When using the generic driver, you must deposit the necessary jar files required by the JDBC driver into the ./lib directory, found inside the AMI installation.

 

After selecting the Generic JDBC driver, you can supply options to direct how the AMI datasource adapter will access the underlying JDBC database.

URL

The url as would be supplied to Java's java.sql.DriverManager.getConnection(…) method.  The ${USERNAME} will be replaced with supplied username and ${PASSWORD} will be replaced with the supplied password

Options

Note that options are key=value pairs that are comma delimited. Use a backslash to escape commas or equal signs within a value.

 

DRIVER_CLASS - fully qualified class name of JdbcDriver, must be available in class path (under lib directory) - required

SHOW_TABLES_CLAUSE - optional - The SQL to execute to retrieve all tables from the datasource (first column is considered). If not supplied, the table selection wizard will be empty

PREVIEW_TABLE_CLAUSE - optional - The SQL to execute to retrieve sample data from a table.  The SQL must contains a ${TABLE} and ${LIMIT} which will be replaced by the table name and a numeric limit.  If not supplied, no sample data will be available for a selected table in the wizard

SUGGGESTED_CLAUSE - optional - The SQL to execute to put in the datamodeler to retrieve data from a table.  The SQL should contain a ${TABLE} which will be replaced by the table name and a numeric limit.  If not supplied, a generic best guess will be automatically generated.

The following example would connect to a mysql database

(The database is on localhost called mydb accessed with myusername and mypassword)

 

Name:

Generic

 

URL:

jdbc:mysql://localhost:3306/mydb?user=${USERNAME}&password=${PASSWORD}

 

Username:

myusername

 

Password:

mypassword

 

Options:

DRIVER_CLASS=com.mysql.jdbc.Driver,SHOW_TABLES_CLAUSE=show tables,PREVIEW_TABLE_CLAUSE=select * from ${TABLE} limit ${LIMIT},SUGGESTED_CLAUSE=select * from ${TABLE} where ${WHERE}

 

Relay to Run on:

<No Relay, Run on Center>
