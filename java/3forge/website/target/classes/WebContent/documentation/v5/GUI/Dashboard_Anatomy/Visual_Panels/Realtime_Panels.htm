Realtime Panels

Realtime panels connect directly to the AMI Realtime Database, meaning that as data is updated in the database, it is immediately pushed to the visualization.

 

Tables

Provides a real-time tabular view of data from the AMI realtime database. Data can be sorted, filtered, and exported by end users. As users click on rows, they can execute relationships or present menu items. Custom columns and filters can be added when in Layout Editor mode. Tables can also be configured to allow edit mode, where users can double click on row(s) to edit select values.

 

Aggregate Tables

Provides a real-time tabular aggregate view of data from the AMI realtime database. Data can be aggregated, sorted, filtered, and exported by end users. As users click on rows, they can execute relationships or present menu items. Custom columns and filters can be added when in Layout Editor mode. Tables can also be configured to allow edit mode, where users can double click on row(s) to edit select values.

 

Heatmaps

Provides a heatmap view of tabular data from the AMI realtime database, where data is grouped and aggregated by one or two levels. Users can zoom in and click on nodes to run relationships.
