Definition

parseDate(String text, String format, String timezone)

Description

Converts a string to a unix epoch time.

Return Value

long(current time in milliseconds UTC since unix epoch) or null

Parameter Definition

String text String to parse
String format Pattern to expect string to be in, see java SimpleDateFormat for options
String timezone Timezone to expect string is in

Examples

long r1 = parseDate("201505030-10:30:40.123","yyyyMMdd-HH:mm:ss.SSS","EST5EDT"); // r1 == 1432996240123
long r2 = parseDate("Jan/03/1982-10:30:40","MMMM/dd/yyyy-HH:mm:ss","UTC"); // r2 == 378901840000
