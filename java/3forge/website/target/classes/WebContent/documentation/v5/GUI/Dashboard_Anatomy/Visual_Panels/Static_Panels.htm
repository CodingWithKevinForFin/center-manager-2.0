Static Panels

A Static Panel renders data from one more more datamodels. It is important to have an understanding of the datamodeler and how to build datamodels before using static visual panels (see Datamodeler tool).

 

Tables

Provides a tabular view of data from a datamodel. The columns can either be predetermined or change dynamically as the underlying data changes. Data can be sorted, filtered, and exported by end users. As users click on rows, they can execute relationships or present menu items. Custom columns can be added when in Layout Editor mode. Tables can also be configured to allow edit mode, where users can double click on row(s) to edit select values.

 

 

Trees

Provides a tree view of data from a datamodel where data is grouped and aggregated up by one or more fields. The tree’s nodes can be expanded/contracted by end users. Data can be sorted, filtered, and exported by end users as well. As users click on rows, they can execute relationships or present menu items. Custom columns can be added when in Layout Editor mode. In the example below, the countries are grouped by continent, and branches can be expanded by clicking the right-pointing arrow as South America has been here. 

 

 

Heatmaps

Provides a heatmap view of tabular data from a datamodel, where data is grouped and aggregated by one or two levels. Users can zoom in and click on nodes to run relationships. In this example, highest level grouping is continent and second level is country.

 

 

2D Charts

Provides a 2D graphical representation of data from one or more datamodels. End users can zoom in and click on elements within the chart to run relationships. Plots, Axes, and Layers can be added when in Layout Editor Mode. AMI supports 10 different types of 2D charts with flexibility to adapt and customize for different use cases.

Plot

One or more plots can be added to a chart, forming a grid of plots. Each plot can contain any number of layers, stacked from front to back such that the front layer will obscure data from layers behind it, depending on the transparency of each layer.

Axis

Any number of horizontal axes can be added to the top and bottom of a column of plots and any number or vertical axis can be added to the left or right of a row of plots. Users can zoom and pan on the axis, effectively controlling the rendering position of all layers attached to it.

XY Layer

A type of layer for plotting data using a Cartesian coordinate system. The layer can be linked to one horizontal and one vertical axis. Users can zoom in and pan XY layers which will update the linked axis, and the zoom state of any other layers linked to the same axis. 

Radial Layer

A type of layer for plotting data using the polar coordinate system.

 

3D Charts

Provides a 3D graphical representation of data from one or more datamodels. There are surface layers for producing smooth surfaces and scatter plots for rendering individual points in 3d space. End users can zoom, rotate and adjust the Field of View. Any number of layers can be added in Layout Editor Mode.

 

Maps

Provides a geographic representation of data from one or more datamodels and using MapBox™ data for the background satellite/street view. End users can zoom.  Users can zoom and pan on the data and select nodes to run relationships. Any number of layers can be added in Layout Editor mode.
