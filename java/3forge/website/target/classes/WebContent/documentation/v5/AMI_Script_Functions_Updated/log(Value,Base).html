Definition

log(number Value, number Base)

Description

Returns the log of a double value in a given base

Return Value

log value, such that in common cases return_value ^ supplied_base = supplied_value

Parameter Definition

number Value value to get log for
number Base base to use when calculating log

Examples

double r1 = log(100,10); // r1 == 2.0
double r2 = log(1000,10); // r2 == 2.9999999999999996
double r3 = log(4,2); // r3 == 2.0
double r4 = log(8,2); // r4 == 3.0
double r5 = log(16,2); // r5 == 4.0
