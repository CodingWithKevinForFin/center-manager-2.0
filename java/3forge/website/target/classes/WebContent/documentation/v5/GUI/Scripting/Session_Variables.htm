Session Variables

These are variables that can be used throughout the dashboard. Session variables can be defined through a variety of means. Variables have a type and value. You can view and edit these variables from Menu > Dashboard > Layout Variables.

Layout Settings

Layouts can defined and saved within a layout file using the Menu > Dashboard > Layout Variables

Properties

Within the AmiWeb (or AmiOne) configuration file you can define session properties using the amiscript.variable.<varname>=<value> notation. See Configuration documentation for details.

Entitlements

Properties can be defined on login using the AmiAuthenticator Plugin. This means that after a user has been successfully granted access to a dashboard, the entitlements server can also pass back variables added to this list of session variables. See AMI Plugins for details.
