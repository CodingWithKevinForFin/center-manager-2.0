Definition

strSubstring(String text, number start, number end)

Description

Get a substring from a string, based on start and end. If start or charsToReplace extend beyond string limits, they will be set to string limits

Return Value

String or null if text

Parameter Definition

String text The base string to get a substring from
number start zero indexed start of substring to return
number end zoro indexed end of substring to return

Examples

String r1 = strSubstring("this",1,2); // r1 == h
String r2 = strSubstring("this",0,4); // r2 == this
String r3 = strSubstring("this",2,100); // r3 == is
String r4 = strSubstring("this",-10,1); // r4 == t
