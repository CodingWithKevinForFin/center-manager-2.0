Function Declaration

While there are many common functions available "out of the box", custom functions can be declared as well. Function declarations follow the same naming convention and scoping rules as variables. The syntax is:

 

 

function declaration  

return_type function_name([param_type  param_name][, param_type param_name ...]){ statement_block }; 

** Evaluates to return_type value 

Examples

{

  Double power(Double n, Integer exponent){

    Double r = n;

     while((exponent--) > 1)

       r *= n;

     return r;

  };

 

  Double alertPower(Double n, Integer exponent){

    session.alert(n + "^" + exponent + " is " + power(n, exponent));

  };

 

  alertPower(2.0, 3);

  alertPower(3.5, 4);

}

 

In this example the power() function is nested inside the alertPower.  Note that the power method has access to the parameters of the function in which it is nested (the n and exponent parameters)

 

{

  Double alertPower(Double n, Integer exponent){

    Double power(){

      Double r = n;

       while((exponent--) > 1)

         r *= n;

       return r;

    };

    session.alert(n + "^" + exponent + " is " + power(n, exponent));

  };

 

  alertPower(2.0, 3);

  alertPower(3.5, 4);

}
