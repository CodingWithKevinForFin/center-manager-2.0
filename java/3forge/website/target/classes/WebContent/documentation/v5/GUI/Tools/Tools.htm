Tools

There are lots of tools available in Editor Mode for helping to build an effective dashboard. These tools are accessed when in editor mode under the Menu > Dashboard menu.

 

Style Manager

 

The style manager is how you can access and edit Style Sets as well as viewing default styles. Access the style manager 

 

Data Modeler

 

The data modeler provides a graphical view of how the various data components and panels work together.

 

Dashboard Objects

 

This provides a single tool for viewing all the objects within the entire dashboard. You can quickly search across all items's configuration as well as view their change histories

 

Defaults Manager

 

The defaults manager provides an easy to use method for determining which panel needs to have defaults set, as well as setting/clearing those defaults.

 

Resource Manager

 

The Resource Manager provides a table for uploading, deleting and viewing resources. Keep in mind these resources are stored inside the AmiCenter
