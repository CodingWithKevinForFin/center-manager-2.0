HTML Panels

Provides a panel where any number of fields can be added. As the panel is scaled, the fields can be designed to automatically react in a variety of ways. 

HTML

The background of the panel can be edited as HTML when in Layout Editor Mode. The HTML can contain references to values from a data model as well.

Fields

Fields can be added, moved, and deleted while in Layout Editor Mode. The field's default value and available choices can be backed by data from a datamodel. The value from a field can be used when calling commands.

The available HTML fields are:

Text Field

Text Fields can be used for input and output.

 

Text Area

Similar to Text Field, but for bigger content which can break to multiple lines. 

 

Numeric Slider

 

Numeric Range

Multi Checkbox

 

Radio Button

Radio Buttons work in groups. When a group of Radio Buttons is formed then user can only make one selection inside the group.

 

Color Picker

 

Color Gradient Picker

Multi Select Field

 

Date Field

Date Field carries a selected date.

 

Date Range Field

Date Range Field carries a start date and an end date.
