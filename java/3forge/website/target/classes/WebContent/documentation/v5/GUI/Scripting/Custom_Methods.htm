Custom Methods

While AMI contains hundreds of predefined functions, layouts can also contain custom methods,  promoting code reuse in favor of code duplication. Custom methods can be used anywhere predefined methods can be used.


Methods can be marked as volatile, meaning they can be overridden by parent layouts. This, effectively, allows for a layout to provide "hooks" that other parent layouts can implement later.
