Creating a Candlestick Chart

In this example we will go through how to create a custom candlestick chart. This type of visualization is very common with price data. Assuming that we are working on some raw or prepared data, we should have similar to the following columns:

 

Time,Open,High,Low,Close

20201023-09:31:00,203.73,203.75,203.29,203.75

 

After attaching the Datasource and creating the Datamodel we select a Scatter Plot chart

 

 

Then we start configuring our plot by selecting Time on the X axis and High on the Y axis

 

 

Next, by clicking on the Advanced tab we are able to make the following options:

 

 

By overriding the Bottom of the Marker with Low variable and making the Markers 1 px thin and square we have created vertical black lines which range from High to Low. The result so far looks like this:

 

 

Let's call our first layer LowToHigh. Now, it's time to add a Layer to our plot. The new one will be OpenToClose

 

 

This time we select Time at X axis, Close at Y axis and Open as Bottom of our Markers. The Markers are wider (e.g., 5 px) and the coloring is conditional on the Formula:

 

Open < Close ? "#5fbf00" : "#ff0000"

 

Optionally, a Border Color for the Markers can be set to make them more distinguishable.
